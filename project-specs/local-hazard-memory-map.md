# Local Hazard Memory Map

## Mission

Build a neighborhood memory map for local hazards: floods, blocked roads, unsafe crossings, heat islands, smoke zones, outages, broken infrastructure, and recurring risks.

## Problem

Local risk knowledge often lives in people’s memories and chat groups. When people move, disasters hit, or officials lack detail, communities lose practical information about what actually happens on the ground.

## MVP

- Offline map notes.
- Hazard categories.
- Time-stamped reports.
- Photo attachments.
- Severity and confidence labels.
- Printable neighborhood summary.
- JSON export for community handoff.

## Technical Plan

- PWA with local storage.
- Map layer with offline fallback.
- GeoJSON export.
- Attachment size limits.
- Optional community review flow later.

## Safety Rules

- Do not expose private home details by default.
- Avoid vigilante or blame-focused reports.
- Make confidence and date visible.
- Separate observed facts from interpretation.

## Portfolio Value

This project demonstrates geospatial UX, offline data capture, community resilience, privacy-aware design, and practical climate adaptation tooling.
