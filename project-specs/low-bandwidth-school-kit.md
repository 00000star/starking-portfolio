# Low-Bandwidth School Kit

## Mission

Build an offline-first education kit for schools, tutors, and learners with unreliable internet, old devices, or limited data.

## Problem

Many education platforms assume constant connectivity, cloud accounts, video streaming, and modern devices. Schools with weak infrastructure need resilient lesson delivery, printable materials, and simple progress tracking.

## MVP

- Static lesson library.
- Printable worksheets.
- Teacher notes.
- Learner progress tracker.
- USB/offline distribution mode.
- Local backup and restore.

## Technical Plan

- Static PWA.
- Markdown or JSON lessons.
- Service worker caching.
- Exportable progress data.
- Print-focused CSS.
- Optional local network sharing later.

## Safety Rules

- Avoid collecting unnecessary student data.
- Keep student progress local by default.
- Make content source and review status visible.
- Support teacher adaptation instead of locking content.

## Portfolio Value

This project demonstrates education technology, offline-first architecture, accessibility, content modelling, and responsible data minimization.
