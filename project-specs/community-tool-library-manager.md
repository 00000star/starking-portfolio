# Community Tool Library Manager

## Mission

Build simple software for community tool libraries, schools, workshops, makerspaces, and repair groups to lend tools safely and maintain them over time.

## Problem

Tool sharing can reduce waste and increase local capability, but groups need borrowing records, maintenance schedules, training notes, safety checks, and repair history without adopting expensive inventory systems.

## MVP

- Tool catalog.
- Borrow and return tracking.
- Maintenance schedule.
- Safety checklist per tool.
- Required training notes.
- Repair history.
- Overdue list.

## Technical Plan

- Static or local-first PWA.
- SQLite or IndexedDB data store.
- QR labels for tools.
- CSV import/export.
- Printable signout sheets.

## Safety Rules

- Flag dangerous tools clearly.
- Require safety acknowledgements for high-risk items.
- Track maintenance before lending.
- Do not store unnecessary borrower data.

## Portfolio Value

This project demonstrates inventory modelling, operations UX, QR workflows, safety-oriented design, and public-good software for circular economies.
