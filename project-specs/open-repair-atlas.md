# Open Repair Atlas

Repository: https://github.com/00000star/open-repair-atlas

Status: MVP built as a standalone offline-friendly repair atlas.

## Mission

Build an open, offline-friendly repair guide for practical devices and tools that people depend on: bicycles, radios, pumps, lights, small engines, and household hardware.

## Problem

Repair knowledge is scattered across videos, forums, manuals, and local experience. Search results often assume stable internet, specific branded parts, and consumer replacement instead of diagnosis, improvisation, and safe reuse.

## MVP

- Repair workflow for bicycles.
- Repair workflow for hand pumps.
- Common failure library.
- Tool and part identification cards.
- Safety warnings for springs, pressure, blades, electricity, fuel, and contaminated parts.
- Printable one-page repair checklists.

## Data Model

- Device category.
- Failure symptoms.
- Likely causes.
- Required tools.
- Required parts.
- Diagnostic steps.
- Repair steps.
- Safety warnings.
- Test procedure.
- Salvage alternatives.

## Technical Plan

- Static PWA.
- JSON repair entries.
- Full-text search.
- Diagram image slots.
- Print-friendly mode.
- Optional offline bundle export.

## Safety Rules

- State when a repair is unsafe for beginners.
- Separate diagnosis from repair action.
- Warn clearly around electrical, fuel, pressure, and sharp hazards.
- Avoid instructions that bypass essential safety controls.

## Portfolio Value

This project demonstrates structured content modelling, accessibility, offline-first UX, technical writing, diagram integration, and practical public-good engineering.
