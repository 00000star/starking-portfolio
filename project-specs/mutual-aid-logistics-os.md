# Mutual Aid Logistics OS

## Mission

Build a lightweight logistics tool for mutual-aid groups, shelters, churches, schools, and neighborhood teams coordinating requests, supplies, volunteers, and deliveries.

## Problem

Small community groups often coordinate with WhatsApp, spreadsheets, paper, and memory. That breaks down when needs grow, volunteers rotate, or accountability matters.

## MVP

- Request and offer board.
- Inventory tracker with quantities and expiry dates.
- Volunteer task board.
- Delivery route notes.
- Beneficiary privacy controls.
- CSV and JSON export.
- Printable daily handoff sheet.

## Technical Plan

- Local-first PWA.
- IndexedDB storage.
- No account required for the first version.
- Optional encrypted export later.
- Optional multi-device sync later.

## Safety Rules

- Minimize personal data by default.
- Make stale information visible.
- Keep audit logs for sensitive inventory changes.
- Avoid exposing vulnerable people’s locations publicly.

## Portfolio Value

This project demonstrates humane operations software, local-first data handling, privacy-aware design, offline workflows, and practical civic-tech product judgement.
