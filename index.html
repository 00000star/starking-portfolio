Featured case study

The Codex: a civilisation operating manual

The Codex started as a tech tree and grew into an offline-first reference system for rebuilding capability from first principles. It is built around the question: what would someone need to know if supply chains, internet access, or Earth itself were unavailable?

Problem Human knowledge is scattered, shallow, and dependency-blind.
Solution Structured technology entries connected by prerequisites, materials, media, hazards, and planners.
Engineering React, TypeScript, JSON schema discipline, generated summaries, PWA caching, and Obsidian export.
Result Live public deployment, offline-capable experience, and a flagship product that anchors the portfolio narrative.
Next step Deeper case studies, richer media, and more high-trust entries with verification-aware content design.
