# 100 Humanity Software Ideas

This catalog expands the portfolio pipeline beyond the first public-good projects. Each idea is intentionally scoped around a real workflow, a neglected user, and a small MVP that can become a serious portfolio artifact.

## Resilience, Disaster, And Climate

1. **Community Cooling Center Finder** - Help cities, libraries, churches, and volunteers publish heat-safe spaces, hours, capacity, water availability, and transit notes. MVP: offline-friendly map, printable list, SMS export, stale-data warnings.

2. **Flood Prep Household Planner** - Help households understand flood risk actions before rain starts. MVP: address-free risk questionnaire, sandbag checklist, document pack list, shutoff instructions, evacuation print sheet.

3. **Wildfire Smoke Safe Room Builder** - Help families create a cleaner indoor room during smoke events. MVP: materials checklist, fan/filter sizing guide, symptom warning list, reminder schedule, source-backed limitations.

4. **Power Outage Food Safety Timer** - Help households track fridge/freezer safety during outages. MVP: outage start timer, food category guide, thermometer log, discard checklist, printable emergency card.

5. **Community Generator Rotation Log** - Help neighborhoods share limited generator power fairly and safely. MVP: device schedule, fuel log, safety warnings, maintenance checklist, handoff records.

6. **Water Point Queue Coordinator** - Help communities manage temporary water distribution. MVP: pickup schedule, household needs, container count, volunteer roster, low-data export.

7. **Shelter Intake Lite** - Help small emergency shelters track beds, needs, allergies, pets, family groups, and departures without a heavy enterprise system. MVP: local-only registry, privacy mode, printable daily roster.

8. **Evacuation Ride Board** - Help neighbors match people needing rides with drivers during evacuations. MVP: request board, capacity fields, accessibility needs, pickup windows, stale request flags.

9. **Community Radio Net Planner** - Help local groups plan radio channels, call signs, check-in times, and message formats. MVP: printable radio plan, message log, role cards, offline guide.

10. **Disaster Rumor Triage Board** - Help coordinators separate verified updates from rumors during emergencies. MVP: claim log, source field, confidence status, last checked time, public summary export.

## Best First Builds From This List

- **Medication Continuity Wallet** because it is small, useful, local-first, privacy-sensitive, and demoable.
- **Tenant Evidence Pack Builder** because it produces a clear user artifact and shows careful legal-adjacent boundaries.
- **Community Pantry Stock Manager** because the workflow is concrete and data modelling is straightforward.
- **Public Meeting Simplifier** because it combines document parsing, civic access, and source-backed summaries.
- **Safety Copy Reviewer** because it can improve every future health, legal, disaster, and AI project in the portfolio.
