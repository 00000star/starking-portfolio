# Portfolio Build Plan

## Goal

Build a portfolio that proves capability through serious projects, not small clones.

## Project Standards

Every portfolio project should eventually include:

- Live demo.
- Clear README.
- Screenshots.
- Architecture diagram.
- Install instructions.
- Test command.
- Build command.
- Short case study.
- Roadmap.
- Security notes if it touches secrets, AI, or user data.

## Flagship Projects

1. The Codex

   - Next: deploy live and add demo video.

2. Obsidian AI Knowledge Engine

3. AI Project Manager

4. AI Safety Secret Scanner
   - Status: planned.

5. Visual Engineering Playground
   - Status: planned.

   - MVP: generate a pulley, lever, wheel-and-axle, and circuit diagram.
